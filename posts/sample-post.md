# My First Blog Post

This is a sample blog post to demonstrate the functionality of this simple Markdown-based blog.

## Section 1: About This Blog

This blog is built using simple HTML, CSS, and JavaScript. It uses the `marked` library to parse Markdown and `highlight.js` for syntax highlighting.

### Subsection 1.1: Features

*   Markdown to HTML conversion
*   Syntax highlighting for code blocks
*   Table of Contents
*   "Copy Markdown" button

## Section 2: Code Example

Here is an example of a JavaScript function:

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet('World'));
```
